# Academic Quarto Website Starter

This is a small Quarto website suitable for a mathematician's personal academic webpage.

## Local preview

Install Quarto, then run:

```bash
quarto preview
```

## Render locally

```bash
quarto render
```

## GitHub Pages deployment

This repository includes a GitHub Actions workflow in `.github/workflows/publish.yml`.

On GitHub, go to:

`Settings` → `Pages` → `Build and deployment` → `Source` → `GitHub Actions`

Then push changes to the `main` branch.
